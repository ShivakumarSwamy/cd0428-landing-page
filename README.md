## Description

A multi-section landing page, with a dynamically updating navigational menu based on the amount of 
content that is added to the page (Source: [Udacity Nanodegree Project Getting Started](https://learn.udacity.com/nanodegrees/nd0011/parts/cd0428/lessons/ls1853/concepts/6131ba9b-bfed-4d51-83a0-4a9df23d5235))

## Usage

- Using for favourite browser or any tool open [index.html](./index.html) or navigate [here](https://shivakumarswamy.github.io/cd0428-landing-page/)
- Observe navigation bar having links to sections in the current page
- Click on any of the section in navigation bar to navigate to that section
- Furthermore, when you are viewing the section, 
  - section will be highlighted with a transparent border and animations
  - in navigation bar section, observe the section you are viewing is highlighted
- You can also add new section in [index.html](./index.html) and observe it is added to navigation bar and to the page too
